## Projeyi Çalıştırma

Proje modern JavaScript modülleri (ES Modules) kullandığından, tarayıcı güvenlik kuralları gereği `index.html` dosyasına çift tıklanarak **açılamaz**. Bir yerel HTTP sunucusu üzerinden açılması gerekmektedir.

### En Kolay Yöntem: `Projeyi_Baslat.bat`

Klasör içindeki **`Projeyi_Baslat.bat`** dosyasına çift tıklayın.  
Python otomatik olarak `http://localhost:8000` adresinde önbelleksiz sunucu başlatır ve tarayıcı otomatik açılır.

> **Gereksinim:** Bilgisayarda Python 3 kurulu olmalıdır. **Ayrıca uygulamanın çalışması için kullanıcı arayüzüne kendi Gemini API anahtarınızı (aistudio.google.com) girmeniz gerekmektedir.**

### Alternatif: VS Code Live Server

VS Code'da `index.html` dosyasına sağ tıklayıp **"Open with Live Server"** seçeneğini kullanabilirsiniz.

## Proje Özeti

Bu proje, B1–B2 seviyesinde İngilizce öğrenen bireylere yönelik geliştirilmiş, yapay zeka destekli bir dil koçu uygulamasıdır. Başlangıçta OpenAI tabanlı olarak geliştirilmiş olup, daha hızlı ve kotası geniş olduğu için sonradan Google'ın son teknoloji **Gemini 1.5 Flash** modeli ile entegre edilmiştir. Uygulama arayüzünden kullanıcı kendi API kodunu girerek projeyi dilediği gibi sınırsızca kullanabilir.

### Modüller

1. **✍️ Writing (Yazma)**  
   Kullanıcı İngilizce bir cümle veya paragraf yazar. Yapay zeka gramer hatalarını tespit eder, Türkçe açıklamalar sunar, düzeltilmiş versiyonu ve daha doğal bir alternatifi gösterir. CEFR seviye tahmini de yapılır.

2. **📖 Reading (Okuma)**  
   10 farklı konudan biri seçilerek yapay zeka, B1 veya B2 seviyesine uygun bir okuma metni üretir. Metin; 4 anlama sorusu, kelime listesi (Türkçe tanımlı) ve TTS (metin sesli okuma) özelliği ile birlikte sunulur.

3. **🎧 Listening (Dinleme)**  
   Yapay zeka doğal bir konuşma diyalogu üretir ve Web Speech API ile seslendirerek kullanıcıya dinletir. 4 hız seçeneği (0.7x – 1.15x) ve 3 anlama sorusu bulunur. Transcript göster/gizle özelliği de mevcuttur.

4. **🎤 Speaking (Konuşma)**  
   Yapay zeka bir konuşma görevi (prompt) üretir. Kullanıcı mikrofona konuşur, Web Speech API sesi metne dönüştürür. Yapay zeka akıcılık, gramer, kelime hazinesi ve telaffuz açısından Türkçe geri bildirim verir.

5. **📚 Vocabulary (Kelime Defteri)**  
   Reading ve Listening modüllerinden kelimeleri "Ekle" butonuyla kelime defterine kaydedilebilir. Flashcard modu ile kaydedilen kelimeler üzerinden interaktif quiz yapılabilir.

6. **📊 Dashboard (İlerleme Paneli)**  
   Kullanıcının toplam kazandığı XP puanları, günlük kullanım serisi (streak), tamamlanan ders sayısı ve haftalık hedef ilerlemesi görüntülenir. Yeni eklenen 14 farklı başarım rozeti de bu alanda gösterilir.

## Teknik Özellikler

- **Mimari:** Tek sayfalı uygulama (SPA), sekme tabanlı navigasyon
- **Modüler Yapı:** Her özellik ayrı bir `.js` dosyasında (ES Modules)
- **Tasarım:** Glassmorphism, karanlık/açık mod, micro-animasyonlar, responsive layout
- **Veri Saklama:** Tüm kullanıcı verileri (XP, streak, kelimeler, geçmiş) `localStorage`'da tutulur
- **Demo Modu:** Kullanıcının API kotası dolduğunda, bağlantısı koptuğunda veya geçersiz API Key girdiğinde uygulama çökmez; ekrana "API Hatası" uyarısı verir ve önceden hazırlanmış gerçekçi demo (mock) veriler devreye girer.
